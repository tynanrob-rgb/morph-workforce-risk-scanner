# Morph Healthcare Risk Scanner

Separate Vite + React project for a healthcare-specific workforce risk scanner. This app is based on the construction scanner structure, but reworked for healthcare staffing pressure, fatigue, musculoskeletal strain, and wellbeing support.

## What it covers

- Healthcare-specific risk inputs and weighting
- Transparent score explanations
- Annual opportunity view if fatigue improves by 10%
- Commercial inputs for operational value, cover pressure, and disruption cost

## Local run

```bash
cd "/Users/tyno/Documents/New project/healthcare-risk-scanner"
npm install
npm run dev
```
